## Image Guidelines

1. **Format**: Use PNG for diagrams, JPG for photographs
2. **Resolution**: Minimum 150 DPI for print, 72 DPI acceptable for web-only
3. **Size**: Keep file sizes under 1MB when possible
4. **Alt text**: All images have alt text specified in the Quarto documents for accessibility
5. **Copyright**: Ensure all images are properly licensed or original creations
